Detailed Description

Instancia del tipo dato MaxStack que nos permite almacenar una serie de valores.

Precondition
Sigue una estructura basada en pila con política LIFO
Internamente sigue una estructura basada en cola, con política FIFO
Se debe añadir #include<maxqueue.h> para su uso
